## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or database required - runs entirely in the browser!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/hostclinic-booking-system.git
```

2. Navigate to the project folder:
```bash
cd hostclinic-booking-system
```

3. Open `index.html` in your browser:
```bash
# On Windows
start index.html

# On Mac
open index.html

# On Linux
xdg-open index.html
```

That's it! No build process or dependencies needed.

## Features in Detail

### Multi-Language Support
- Seamless switching between English and Arabic
- RTL (Right-to-Left) support for Arabic
- All UI elements translated
- Language preference saved in browser

### Appointment Booking Flow
1. User logs in or registers
2. Selects department
3. Chooses doctor from available specialists
4. Picks date and time slot
5. Provides reason for visit
6. Confirms booking
7. Views appointment in "My Appointments"

### Admin Features
- View all appointments with filtering
- Manage patient records
- Add/remove doctors with photos
- Configure doctor schedules and availability
- Cancel appointments if needed

### Responsive Design
- Mobile-friendly navigation
- Touch-optimized controls
- Adaptive layouts for all screen sizes
- Smooth animations and transitions
